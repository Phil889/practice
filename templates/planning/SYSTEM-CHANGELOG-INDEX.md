# System Changelog — HSI Index

**Purpose:** one-line-per-HSI lookup so the supervisor can reason about hypothesis history without loading every detailed entry. The supervisor reads this file FIRST on every invocation; only opens detailed `SYSTEM-CHANGELOG.md` entries (or `_archive/hsi/<quarter>.md` archived entries) when reasoning about a specific hypothesis.

**Maintained by:** the supervisor. Updated automatically on:
- Every `mode: hygiene` run (when an HSI moves to archive)
- Every HSI status flip (VERIFIED / REFUTED / REGRESSED)
- Every new HSI proposal

**Format:** strict tab-aligned table for fast grep. Don't reformat into prose; the supervisor's parser depends on it.

## Status legend

- **PROPOSED** — hypothesis written; not yet implemented. The next step is to apply the change and flip to APPLIED-PENDING.
- **APPLIED-PENDING-VERIFICATION** — change implemented; awaiting evidence from the next supervisor pass. Always active in SYSTEM-CHANGELOG.md (never archive).
- **APPLIED-VERIFIED** — supervisor confirmed the hypothesis via the verification probe; remains active for 14 days + ≥3 supervisor passes before archive eligibility.
- **REFUTED** — supervisor's verification probe disconfirmed the hypothesis. Replacement HSI must be drafted before archive.
- **REGRESSED — superseded by HSI-NNN** — was VERIFIED, then a later run showed regression. The successor HSI's APPLIED-VERIFIED is the archive trigger.
- **ARCHIVED** — moved to `_archive/hsi/<quarter>.md`. Path field points to the archive file; supervisor opens it only when reasoning about that specific hypothesis.

## Why this index exists

The full `SYSTEM-CHANGELOG.md` grows linearly with the harness's lifetime. Without an index, the supervisor reads ~30K tokens of historical hypothesis just to answer "did we ever try X?" The index keeps that lookup at <2K tokens — supervisor's working memory stays under budget per the [HYGIENE-POLICY.md](_context/HYGIENE-POLICY.md).

**Don't edit this file by hand.** It's regenerated by the supervisor on hygiene runs. Manual edits get overwritten.
